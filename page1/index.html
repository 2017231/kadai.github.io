page1 - html

htmlとは、Hyper Text Markup Languageの略称。

"htmlタグ"を用いて文章を構造化する。

htmlの書き方

<で始まる開始タグ、</で始まる"終了タグ"によって"コンテンツ"を挟む。これが一つの要素。

<html>

  <head>
      コンテンツ
  </head>
              
  <body>
    コンテンツ
  </body>
              
</html>

というような場合、headタグ、bodyタグは親要素であるhtmlタグの子要素となる。
