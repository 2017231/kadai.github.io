page2 - CSS

CSSとは、Cascading Style Sheetsの略称で、ウェブページのスタイルを指定する言語。

htmlと組み合わせて使用する。

CSSの基本的な書き方

CSSは3種類の要素の組み合わせでできる。

  • セレクタ (変更する対象)
  • プロパティ　(変更内容)
  • 値 (変更内容の具体的な値)
h1 {
    font-size: 18px:
}

というような場合

[h1] - セレクタ

[font-size] - プロパティ　後ろに[ : ]を入れる

[18px] - 値　後ろに[ ; ]を入れる

ということになる。
